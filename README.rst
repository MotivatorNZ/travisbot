============
 Travis Bot
============

A bot for Discord. (Work in progress)

Development
===========

To setup and run the basic steps of the bot.

Installation
------------

.. code-block:: console

    $ python -m venv .
    $ . bin/activate
    (travisbot)$ pip install -e .[fast]

Running
-------

.. code-block:: console

    (travisbot)$ python -m travisbot
